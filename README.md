## 🌿 About This Project

> A comprehensive security and compliance checklist designed to help teams audit their AWS cloud environment against **security best practices** and **EU GDPR regulations**.

Whether you're a cloud engineer, security analyst or student — this checklist gives you a clear framework to assess and improve your cloud security posture.

---

## 📋 What's Covered

| Section | Description |
|---|---|
| 🔑 IAM | Identity & access management controls |
| 🔐 Encryption | Data protection at rest and in transit |
| 🌐 Network Security | VPC, WAF, Shield and subnet configuration |
| 📡 Monitoring | Logging, alerting and threat detection |
| 📋 GDPR | Data privacy and compliance requirements |
| 🏗️ Architecture | AWS best practices and resilience |

---

## 🚀 How To Use

1. Open the [`CHECKLIST.md`](./CHECKLIST.md) file
2. Work through each section
3. Tick off items as your environment meets each requirement
4. Use the **Audit Log** at the bottom to track reviews over time
